# 电视直播源自动更新

这个项目自动从 tonkiang.us 抓取电视直播源，并测试其速度和有效性。

## 功能特点

- 🔄 **自动更新**: 每天自动运行，更新直播源
- 🚀 **速度测试**: 测试每个直播源的响应速度
- 📊 **智能排序**: 按速度从快到慢排序
- 🎯 **分类整理**: 按央视、卫视、凤凰、香港等分类
- 📁 **多格式输出**: 同时生成 M3U 和 TXT 格式文件
- ⚡ **多线程处理**: 快速处理所有频道

## 生成的文件

- `tv_sources.m3u` - 所有频道的合并 M3U 文件
- `tv_sources.txt` - 所有频道的合并 TXT 文件
- `tv_sources_央视.m3u/txt` - 央视频道
- `tv_sources_卫视.m3u/txt` - 地方卫视
- `tv_sources_凤凰.m3u/txt` - 凤凰卫视
- `tv_sources_香港.m3u/txt` - 香港电视台
- `STATS.md` - 统计信息

## 使用说明

1. 将此仓库 Fork 到你的 GitHub 账户
2. 默认会每天自动运行
3. 你也可以在 Actions 页面手动触发
4. 生成的直播源文件在根目录

## 支持的频道

包括央视全系列、各地方卫视、凤凰卫视、香港电视台等50+个频道。
